## 2.3 字符串
字符串就是一系列字符
### 2.3.1 使用方法修改字符串的大小写
#### 1. .title()方法
方法title()以首字母大写的方式显示每个单词
```python
name = "ada lovelace"
>>>print(name.title())

Ada Lovelace
```
#### 2. .upper()方法
方法upper()可以将字符串改为全部大写或全部小写
```python
name = "ada lovelace"
>>>print(name.upper())

ADA LOVELACE
```
#### 3. .lower()方法
方法lower()可以将字符串改为全部小写
```python
name = "ADA LOVELACE"
>>>print(name.lower())

ada lovelace
```
### 2.3.2 在字符串中使用变量
在字符串中使用变量的值
- 例子:
```python
first_name = "ada"
last_name = "lovelace"
full_name = f"{first_name}{last_name}"
>>>print(full_name)

ada lovelace
```
- 如何使用:
1. 在前引号前加上字母f
2. 再将要插入的变量放在{}内
- 注意:
此方法等效为.format()方法
### 2.3.3 使用制表符或换行符来添加空白
#### 1. 添加制表符
使用字符组合\t
```python
>>>print("python)
>>>print("\tpython")

python
    python
```
#### 2. 添加换行符
```python
>>> print("Languages:\npython")

Languages:
python
```
