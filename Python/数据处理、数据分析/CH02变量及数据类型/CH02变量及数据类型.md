#### 1. Numbers数字
Numbers数字分为:int整型,long长整型,float浮点型,complex复数
- 例子:
```python
x1 = 10
x2 = 10.0
print(x1,x2)
print(type(x1),type(x2))
```
```python
10 10.0
<class 'int'> <class 'float'>
```
